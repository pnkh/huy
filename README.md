# Nhiếp Ảnh Viễn Đông Gallery

Bộ file web tĩnh để đưa lên GitHub Pages.

## File chính

- `index.html`: giao diện gallery chính.
- `.nojekyll`: giúp GitHub Pages phục vụ file tĩnh trực tiếp.

## Cách publish nhanh

1. Tạo repository mới trên GitHub.
2. Upload `index.html` và `.nojekyll` vào repository.
3. Vào Settings → Pages.
4. Chọn Source: Deploy from a branch.
5. Chọn Branch: main, Folder: /root.
6. Save và chờ GitHub Pages tạo link.
